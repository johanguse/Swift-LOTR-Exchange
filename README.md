- SwiftUI 3
- Swift 5.5
- Xcode 13
- Swift Types (String, Double, Int, and more)

- App Icons and Image Assets
- Project-Based Approach
- Build complete apps from scratch
- Essential SwiftUI layout tools (Grids, Stacks, and more)
